# Signup Wizard Illustration Brief — AI Image Generation Scripts

This document contains detailed prompts for generating illustrations that align with each step of the signup wizard. Each prompt is optimized for AI image generation tools (Midjourney, DALL-E, Stable Diffusion, etc.) and designed to produce consistent, on-brand visuals.

## Art Direction Notes

- **Style:** Flat vector illustration with soft gradients, friendly and approachable, modern SaaS aesthetic
- **Color palette:** Primary blues and teals (`#0EA5E9`, `#06B6D4`), warm accent coral (`#F97316`), neutral grays for backgrounds
- **Characters:** Diverse small-business owners (retailers, shopkeepers) in casual-professional attire. No generic stock-photo faces — stylized, inclusive, friendly
- **Mood:** Empowering, clear, optimistic — never corporate-cold
- **Backgrounds:** Clean, minimal, with subtle depth. Avoid clutter.
- **Aspect ratio:** 16:9 for step backgrounds, 1:1 for icons/spot illustrations

## Step 1 — Mission Hook
**Title:** "Big-Box Retailers Built This for Themselves. Now It's Yours."

**Prompt:**
> A stylized flat-vector illustration of a confident small-business retailer standing in front of their charming local store, looking toward a massive distant big-box warehouse on the horizon. A glowing digital bridge made of light and data flows from the retailer's POS tablet directly up into Google search results, a directory map, and a beautiful storefront page floating in the sky. The retailer is smiling, empowered, holding a Clover POS device. Soft blue and teal gradients, warm coral accent on the store awning. Clean minimal background, modern SaaS illustration style, optimistic and empowering mood. No text. 16:9.

**Alternative / Simpler prompt:**
> Flat vector illustration, small retail shop owner looking empowered with a tablet showing inventory data streaming up into Google search icons and a storefront page. Big warehouse silhouette in distant background. Blue, teal, coral palette. Clean minimal SaaS style. 16:9.

## Step 4 — Commitment

## Brand Consistency Checklist

When generating images, verify:
- [ ] No text embedded in illustrations (let the UI layer handle copy)
- [ ] Diverse, friendly characters when people are shown
- [ ] Clover POS device visible in steps where POS integration is a key message
- [ ] Google visual language (Search, Maps, Shopping) used in Discovery step
- [ ] Payment/delivery iconography used appropriately in E-commerce and Omnichannel
- [ ] Shield/checkmark iconography used for deposit/guarantee in Commitment
- [ ] All images share the same flat-vector, minimal, SaaS-friendly style
- [ ] Color palette stays within blue-teal-coral range

## Recommended Generation Settings

| Tool | Recommended Settings |
|---|---|
| **Midjourney** | `--ar 16:9 --style raw --s 250` for backgrounds; `--ar 1:1 --style raw --s 150` for icons |
| **DALL-E 3** | Use "flat vector illustration, minimal, SaaS style, no text" in every prompt |
| **Stable Diffusion XL** | Use a flat-vector LoRA if available; CFG 7-8; steps 30-40 |
| **Ideogram** | Excellent for consistent style across a set; use the same style reference seed |

## Usage in Code
