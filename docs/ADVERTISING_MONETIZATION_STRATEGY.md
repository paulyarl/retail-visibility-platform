# Advertising & Monetization Strategy: Multi-Revenue Model

**Document Owner:** Product & Revenue Team  
**Date:** 2025-11-06  
**Status:** Strategic Framework

---

## Executive Summary

By introducing **strategic advertising placements** on lower-tier storefronts, the platform can:

1. **Generate additional revenue** ($50K-200K+ annually)
2. **Subsidize lower-tier pricing** (keep entry price low)
3. **Promote platform upgrades** (self-serve marketing)
4. **Enable Google AdSense revenue** (passive income)
5. **Create upgrade incentive** (ad-free = premium feature)

**Key Principle:** Ads should be tasteful, non-intrusive, and provide valueâ€”not degrade the user experience.

## Revenue Model Evolution

### **Current Model (Subscription Only)**
```
Revenue Streams:
1. Tier subscriptions ($29-999/mo)

Limitations:
- Single revenue source
- Price-sensitive customers churn
- No way to subsidize entry tiers
- Leaves money on the table
```

### **Enhanced Model (Subscription + Advertising)**
```
Revenue Streams:
1. Tier subscriptions ($29-999/mo)
2. Platform ads (upgrade promotions)
3. Google AdSense (passive income)
4. Sponsored placements (future)

Benefits:
- Multiple revenue sources
- Lower CAC (can subsidize entry tier)
- Higher LTV (more monetization paths)
- Self-serve upgrade marketing
- Passive income from traffic
```

## Ad Placement Guidelines (UX Best Practices)

### **Principles:**

1. **Non-Intrusive**
   - Ads should not block content
   - No pop-ups or interstitials
   - Dismissible banners only

2. **Tasteful Design**
   - Match storefront aesthetic
   - Professional appearance
   - Clear labeling ("Ad" or "Sponsored")

3. **Mobile-Optimized**
   - Responsive ad units
   - Appropriate sizing for mobile
   - No accidental clicks

4. **Performance-Conscious**
   - Lazy loading
   - Async script loading
   - No impact on Core Web Vitals

5. **Privacy-Compliant**
   - GDPR/CCPA compliant
   - Cookie consent
   - User opt-out options

### **Google AdSense Component**

```typescript
// components/storefront/AdSenseUnit.tsx
'use client';

import { useEffect, useRef } from 'react';

interface AdSenseUnitProps {
  adSlot: string;
  adFormat?: 'auto' | 'rectangle' | 'vertical' | 'horizontal';
  fullWidthResponsive?: boolean;
  className?: string;
}

export function AdSenseUnit({
  adSlot,
  adFormat = 'auto',
  fullWidthResponsive = true,
  className = '',
}: AdSenseUnitProps) {
  const adRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    try {
      // Load AdSense script if not already loaded
      if (!window.adsbygoogle) {
        const script = document.createElement('script');
        script.src = 'https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js';
        script.async = true;
        script.crossOrigin = 'anonymous';
        script.setAttribute('data-ad-client', process.env.NEXT_PUBLIC_ADSENSE_CLIENT_ID || '');
        document.head.appendChild(script);
      }

      // Push ad to AdSense queue
      if (window.adsbygoogle && adRef.current) {
        (window.adsbygoogle = window.adsbygoogle || []).push({});
      }
    } catch (error) {
      console.error('AdSense error:', error);
    }
  }, []);

  return (
    <div className={`adsense-container ${className}`}>
      <div className="text-xs text-gray-400 mb-1 text-center">Advertisement</div>
      <ins
        ref={adRef}
        className="adsbygoogle"
        style={{ display: 'block' }}
        data-ad-client={process.env.NEXT_PUBLIC_ADSENSE_CLIENT_ID}
        data-ad-slot={adSlot}
        data-ad-format={adFormat}
        data-full-width-responsive={fullWidthResponsive.toString()}
      />
    </div>
  );
}
```
