# Chain Management Callout Section
**Component to add to features page**

Insert this section before the "Powerful Features for Modern Retailers" section.

This creates a comprehensive callout showcasing all 8 propagation types with individual cards, similar to the Quick Start Wizard and Product Scanning callouts.

## Section includes:
1. Header with "ENTERPRISE CHAIN MANAGEMENT" badge
2. 8 propagation type cards in 4x2 grid:
   - Products/SKUs (ACTIVE)
   - Categories (ACTIVE)
   - GBP Category Sync (ACTIVE)
   - Business Hours (COMING SOON)
   - Business Profile (COMING SOON)
   - Feature Flags (COMING SOON)
   - User Roles & Permissions (COMING SOON)
   - Brand Assets (COMING SOON)
3. Value proposition footer with stats

Each card shows:
- Icon
- Title
- Status badge (ACTIVE or COMING SOON)
- Description
- 3 key benefits with checkmarks

The section uses emerald/teal gradient background to match the chain management theme.
